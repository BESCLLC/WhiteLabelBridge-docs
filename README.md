- go-live checklist

## Product Summary

WhiteLabelBridge is a white-label bridge platform that lets token teams connect:
- one external chain (`ETH/BSC/POL/ARB/OP/custom EVM` or `Solana`)
- `BESC Hyperchain`

Core model:
- setup wizard driven configuration
- exact decimal-safe conversion logic
- selectable `vault` or `mint` execution modes per side
- one-command server bootstrap flow
