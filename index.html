Public Documentation for WhiteLabelBridge

WhiteLabelBridge gives token teams a full bridge platform between their home chain and BESC Hyperchain, with a guided setup wizard and production-safe decimal handling.

Start Deployment Guide Open Wizard Reference

What This Product Delivers

One External Chain + BESC

Connect Ethereum, BSC, Polygon, Arbitrum, Optimism, Solana, or custom EVM to BESC Hyperchain.

Mode Per Side

Run vault (lock/release) or mint (burn/mint) independently on each side.

Install + Wizard Flow

Deploy with one command, complete wizard, finalize, and go live with PM2-managed services.

Exact Decimal Safety

Cross-decimal conversion rejects non-exact outcomes to prevent over mint, under mint, or drift.

Fast Path

  1. Run npm run setup:server on your server.
  2. Open setup wizard URL printed in terminal.
  3. Provide chain data, signer key, contract addresses, and exact decimals.
  4. Run npm run setup:finalize to apply runtime and infrastructure settings.
  5. Execute full go-live checklist before opening the bridge publicly.
BESC queue releases are fixed at 1000 gwei. External EVM routes use chain-native gas estimation. Solana routes use native Solana fees.
